# Cut Plan — Summer 2025

Personal fitness tracking site for an 18-week cut plan.

## Deploying to GitHub Pages

1. Create a new **public** repository on GitHub (e.g. `cut-plan-2025`)
2. Upload `index.html` to the repo root
3. Go to **Settings → Pages**
4. Under "Source", select **Deploy from a branch**
5. Choose **main** branch, **/ (root)** folder
6. Click Save — your site will be live at `https://yourusername.github.io/cut-plan-2025`

That's it. No build step, no dependencies, no server required.

## Making updates

Edit `index.html` directly in the GitHub web editor (pencil icon) or push changes from your local machine. GitHub Pages redeploys automatically within ~30 seconds.

## What's in the site

- **Overview** — cut plan phases, macros by phase, training, supplements, key rules
- **Meal Plan** — weekly schedule for all 3 rotation weeks with macro tables and batch quantities
- **Recipes** — full ingredient quantities and steps for every meal and snack
- **Grocery** — interactive checklists that save your checked state in the browser
